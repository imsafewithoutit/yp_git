Заголовки разных уровней создают решётками:

    # H1 — заголовок первого уровня, самый большой
    ## H2 — заголовок второго уровня, поменьше
    ### H3
    #### H4
    ##### H5
    ###### H6 — заголовок шестого уровня, самый маленький 



Разделительную линию между абзацами - тремя тире:

    ---



Разрыв строки - двумя пробелами (в примере они обозначены точками ⋅⋅), или с помощью <br>:

    Текст до переноса⋅⋅  
    Текст после переноса <br>
    Текст после второго переноса 



Новый параграф - отделяется двумя символами переноса в конце предыдущей строки (т.е. 2 раза нажать Enter). Если один символ переноса, и не поставить два пробела или <br> - то текст сольётся в одну строку.



Курсивом - через звёздочки или нижние подчёркивания. Полужирный шрифт - через двойные звездочки или двойные нижние подчеркивания. Полужирный курсив - через совмещение звездочек/подчеркиваний:

    *курсив* или _курсив_
    **полужирный** или __полужирный__
    **полужирный _курсив_** 

Зачеркнутый - через двойные тильды:

    ~~Зачёркнутый текст~~ 



Нумерованные списки - в начало строки поставить цифру с точкой:

    1. Первый пункт нумерованного списка.
    2. Второй пункт. 

Ненумерованный список - в начало строки поставить звездочку с пробелом или дефис с пробелом:

    * первый пункт ненумерованного списка;
    * второй пункт ненумерованного списка

    - первый пункт ненумерованного списка;
    - второй пункт ненумерованного списка 



Ссылка - текст заключить в квадратные скобки, а после него без пробела адрес в круглые скобки:

    [Яндекс](https://www.yandex.ru) 

Если нужна всплывающая подсказка для ссылки при наведении - ее указывают в кавычках внутри скобок после адреса через пробел:

    [Яндекс](https://www.yandex.ru "Я Yandex!") 



Код - заключить в тройные косые кавычки, а после открывающих кавычек указать язык на котором написан код (для поддержки синтаксиса):

    ```bash
    ls - la
    ```
    ```html
    <h1>А я просто текст</h1>
    ```
